# ✈️ Plann.er

O **Plann.er** é uma aplicação completa para o planejamento de viagens em grupo. Com ele, você pode organizar seu roteiro, convidar amigos, gerenciar atividades e links importantes, tudo em um só lugar.

Este projeto foi desenvolvido durante o evento **NLW Journey** da [Rocketseat](https://www.rocketseat.com.br/).

---

## 📁 Estrutura do Projeto

O repositório é dividido em dois subprojetos principais:

| Pasta | Descrição | Tecnologias Principais |
|---|---|---|
| [**api**](./api) | Back-end da aplicação | Fastify, Prisma, Zod, PostgreSQL |
| [**web**](./web) | Front-end da aplicação | React 19, Vite, Tailwind CSS 4, TanStack |

---

## 🚀 Como Começar

### Pré-requisitos

Antes de iniciar, você vai precisar ter instalado:
- [Node.js](https://nodejs.org/) (recomendado v22+)
- [pnpm](https://pnpm.io/) (gerenciador de pacotes)
- [Docker](https://www.docker.com/) (para rodar o banco de dados)

### Passos Rápidos

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/seu-usuario/plann.er.git
   cd plann.er
   ```

2. **Configure e rode a API:**
   Siga as instruções detalhadas no [**README da API**](./api/README.md).

3. **Configure e rode o Web:**
   Siga as instruções detalhadas no [**README do Web**](./web/README.md).

---

## 🛠️ Tecnologias Utilizadas

Este projeto utiliza o que há de mais moderno no ecossistema JavaScript/TypeScript:

- **Back-end:** Fastify, Prisma ORM, Zod, Nodemailer.
- **Front-end:** React 19, Tailwind CSS 4, TanStack (Router, Query), Lucide React.
- **Padronização:** ESLint, Prettier, TypeScript.
